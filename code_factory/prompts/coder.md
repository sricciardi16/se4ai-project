Your task is to implement a specific programming task by writing or modifying a source code file.

# Input

You will be provided with:
1. **Task Description:** The exact instructions detailing what must be built, modified, or tested.
2. **Current File Content:** The existing source code of the target file. (If the file does not exist yet, this will be empty).

# Output

Produce the final, complete source code for the target file that fulfills the Task Description.

**Requirements:**

- **Completeness:** You must return the *entire* file. If you are modifying an existing file, you must integrate your changes and return the complete file from the first line to the last.
- **No Placeholders:** You are strictly forbidden from using placeholders, pseudo-code, or truncation (e.g., do not use `# ... existing code ...`). 
- **Execution Ready:** The resulting code must be fully functional, syntactically correct, and include all necessary imports.

## Output Format

Output the complete source code wrapped in a single Markdown code block. 

```[language]
[Complete, unmodified original code combined with your exact changes]
```
